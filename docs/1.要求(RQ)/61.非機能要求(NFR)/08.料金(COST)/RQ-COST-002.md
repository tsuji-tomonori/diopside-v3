## SnowCard（日本語）
- 要求ID: RQ-COST-002
- 種別: 非機能要求
- 優先度: MUST
- 要求: 本システムは、AWS Budgetsによるコスト閾値アラートを自動化し、予算超過を早期検知すること。
- 根拠: [[RQ-COST-001]] の月額3,000円上限を遵守するため、段階的な警告で手動介入のタイミングを確保する必要があるため。
- 受入基準:
  - AWS Budgetsで月額予算が設定され、50%・75%・90%の閾値でアラートが発報される。
  - アラート通知先としてSNSトピックが設定され、メール通知が届く。
  - Budgets設定はCDKで管理され、手動設定に依存しない。
  - アラート発報から48時間以内に抑制施策の要否を判断し記録する。
- 例外/エラー:
  - AWS Budgets自体の障害でアラートが発報されなかった場合は、月次コストレビューで検知し、代替手段を検討する。
  - 90%閾値超過後に抑制施策が間に合わず上限超過した場合は、[[RQ-COST-001]] の例外手順に従う。
