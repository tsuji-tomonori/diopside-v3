## SnowCard（日本語）
- 要求ID: RQ-DEV-008-03
- 種別: 非機能要求
- 優先度: MUST
- 要求: 本システムは、DADSを踏まえたサービス固有のスタイルガイドを作成・維持できる。
- 根拠: DADSは共通基盤であり、サービス文脈に応じた再定義を前提としている。独自スタイルガイドがない場合、画面横断で表現が揺れてアクセシビリティ判断基準が統一できないため、継続運用の規範として必要である。
- 受入基準:
  - 本システムは、DADSを踏まえたサービス固有のスタイルガイドを作成・維持できる。
- 例外/エラー:
  - 例外条件は個別要求に明示されない限り適用しない運用ができる。

  - [[RQ-RDR-052]]
