## SnowCard（日本語）
- 要求ID: RQ-UX-008-02
- 種別: 非機能要求
- 優先度: MUST
- 要求: ターゲット領域はボタン44x44、ボタン以外のリンク要素24x24 CSS px以上を確保できる。
- 根拠: 操作対象が小さいと誤タップが増加し、モバイル利用や運動制約がある利用者で操作不能を招く。DADSはボタン44x44 CSS px以上、リンク要素24x24 CSS px以上を指針としているため、要素種別で下限を分けて規定する必要がある。
- 受入基準:
  - ターゲット領域はボタン44x44、ボタン以外のリンク要素24x24 CSS px以上を確保できる。
- 例外/エラー:
  - 例外条件は個別要求に明示されない限り適用しない運用ができる。
- 依存・関連:
  - [[RQ-RDR-049]]
  - [[RQ-RDR-052]]
