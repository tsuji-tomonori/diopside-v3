## SnowCard（日本語）
- 要求ID: RQ-UX-021
- 種別: 非機能要求
- 優先度: MUST
- 要求: 本システムは、CI/CDでアクセシビリティ検査を自動実行し、重大違反をリリースゲートで遮断すること。
- 根拠: 手動確認漏れによる品質低下を防止するため。
- 受入基準:
  - Pull Requestごとに自動アクセシビリティ検査を実行する。
  - 重大違反（Critical/High）が1件以上の場合はマージ不可とする。
  - 検査結果を履歴として90日以上保持する。
- 例外/エラー:
  - 誤検知を一時除外する場合は、有効期限付き例外設定と改善チケットを必須化する。
