## SnowCard（日本語）
- 要求ID: RQ-UX-008
- 種別: 非機能要求
- 優先度: MUST
- 要求: 本システムは、ポインター操作対象のタップ領域を十分なサイズで提供すること。
- 根拠: モバイル端末での誤操作を低減するため。
- 受入基準:
  - 主要操作ボタン/リンクのターゲットサイズを最小24x24 CSS px以上とする。
  - 隣接ターゲット間は誤タップ防止のため十分な間隔を確保する。
- 例外/エラー:
  - テキスト内インラインリンクは例外とするが、拡大しても選択可能であること。
