## SnowCard（日本語）
- 要求ID: RQ-UX-008
- 種別: 非機能要求
- 優先度: MUST
- 要求: 本システムは、ポインター操作対象のタップ領域を十分なサイズで提供できる。
- 根拠: モバイル端末での誤操作を低減するため。
- 受入基準:
  - 主要操作ボタン/リンクのターゲットサイズを最小24x24 CSS px以上にできる。
  - 隣接ターゲット間は誤タップ防止のため十分な間隔を確保できる。
- 例外/エラー:
  - テキスト内インラインリンクを例外扱いする場合でも、拡大時に選択可能な状態を維持できる。
- 依存・関連:
  - [[RQ-RDR-018]]
  - [[BD-APP-UI-002]]
