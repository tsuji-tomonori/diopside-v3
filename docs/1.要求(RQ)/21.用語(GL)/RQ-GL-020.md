## 定義
| 項目 | 内容 |
|---|---|
| 用語ID | `RQ-GL-020` |
| 用語名 | [[RQ-GL-020|運用サイクル]] |
| 英名 | `operation_cycle` |
| 定義 | 収集・確認・公開反映・判定記録を1巡させる、日次運用の反復単位。 |
| 判定条件/適用範囲 | 公開アーカイブの新着反映、運用判定、受入判定の更新頻度を評価する文脈で適用する。 |

## 利用ルール
- 本文では [[RQ-GL-020|運用サイクル]] を使用し、曖昧な期間表現（例: 定期運用、運用タイミング）を避ける。
- 既定の周期は日次（24時間以内）とし、例外運用時は理由と実施日時を記録する。
