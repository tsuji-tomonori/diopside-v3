## 定義
| 項目 | 内容 |
|---|---|
| 用語ID | `RQ-GL-015` |
| 用語名 | [[RQ-GL-015|盛り上がり区間]] |
| 英名 | `highlight_segment` |
| 定義 | コメント密度が相対的に高いと判定された連続時間区間。動画詳細で見どころ導線として表示し、クリック時に該当秒へ遷移する単位。 |
| 判定条件/適用範囲 | 動画詳細のコメント密度可視化で、区間の開始秒・終了秒・密度指標・コメント件数を持つ表示/遷移単位として適用する。 |

## 利用ルール
- 区間は `開始秒 < 終了秒` を満たすこと。
- 区間クリック時の遷移秒は開始秒を用いる。
- 文書・実装・テストで同じ意味で使用する。
