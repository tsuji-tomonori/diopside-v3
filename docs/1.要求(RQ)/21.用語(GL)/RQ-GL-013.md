## 定義
| 項目 | 内容 |
|---|---|
| 用語ID | `RQ-GL-013` |
| 用語名 | [[RQ-GL-013|タグ種別]] |
| 英名 | `tag_type` |
| 定義 | タグを利用者に表示・選択させる際の分類単位。各タグは必ず1つの[[RQ-GL-013|タグ種別]]に属する。 |
| 判定条件/適用範囲 | 検索UIのカテゴリ表示、タグ選択ルール、[[RQ-GL-005|タグ辞書]]配信契約で適用する。 |

## 画面構成で確認した種別
- `チャンネル`
- `大分類`
- `出演者名`
- `ユニット名`
- `ゲーム名`
- `ゲームジャンル`
- `シリーズ名`
- `コンテンツ名`
- `コンテンツジャンル`
- `企画名`
- `企画大分類`
- `企画ジャンル`
- `配信ジャンル`
- `動画形式`

## 利用ルール
- 表示順は種別定義の順序に従う。
- 単一選択種別（`チャンネル`、`大分類`、`企画大分類`、`動画形式`）は1動画につき1値とする。
- 必須種別（`チャンネル`、`大分類`、`動画形式`）は欠落を許容しない。
