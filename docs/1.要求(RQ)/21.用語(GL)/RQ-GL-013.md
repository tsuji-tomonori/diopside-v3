## [[RQ-GL-013|タグ種別]]一覧
| [[RQ-GL-013|タグ種別]] | 説明 | 付与ルール |
|---|---|---|
| `チャンネル` | 投稿元チャンネル。 | 投稿元チャンネルを1つ付与する。 |
| `大分類` | 動画/配信の主カテゴリ。 | タイトル/概要欄・既存タグから主内容を1つ選ぶ。 |
| `出演者名` | 出演者（配信者/投稿者含む）。 | 概要欄やタイトルに基づき、出演者を付与する（誤記は正規化）。投稿元チャンネルも出演者として扱う。 |
| `ユニット名` | 固定ユニット・グループ名（例: Crossick、フルトイ）。 | 固定ユニット名が明示される場合に付与する。 |
| `ゲーム名` | 実況/プレイしたゲームの正式名称。 | タイトル/概要欄からゲームの正式名称を採用する（略称は公式表記へ正規化）。 |
| `ゲームジャンル` | Steamタグ相当のゲームジャンル。 | ゲーム名に基づき、Steamタグ相当のジャンルを最低1つ付与する。 |
| `シリーズ名` | シリーズ/フランチャイズ名。 | シリーズ作品の場合、シリーズ名を付与する。 |
| `コンテンツ名` | 同時視聴の対象コンテンツ名。 | 同時視聴の対象をタイトル等から特定して付与する。 |
| `コンテンツジャンル` | 同時視聴の対象ジャンル（映画/アニメ/ドラマ/イベント等）。 | 対象に合わせてジャンルを統一して付与する。 |
| `企画名` | 企画の名称（例: 初配信、お披露目配信）。 | 企画の名称を付与する（同種企画は名称を統一）。 |
| `企画大分類` | 企画の大分類（トーク/ライブ/声劇/バラエティー等）。 | 企画内容に応じて大分類を付与する。 |
| `企画ジャンル` | 企画のジャンル（企画内で統一）。 | 企画名ごとにブレない粒度でジャンルを付与する。 |
| `配信ジャンル` | 配信の特徴（マシュマロ、晩酌、耐久、コラボ等）。 | 必要に応じて追加する（過度に細かいものは避ける）。 |
| `動画形式` | 動画の形式（配信/動画）。 | 動画形式として「配信」または「動画」のどちらかを必ず1つ付与する。 |

## 利用ルール
- 表示順は種別定義の順序に従う。
- 単一選択種別（`チャンネル`、`大分類`、`企画大分類`、`動画形式`）は1動画につき1値とする。
- 必須種別（`チャンネル`、`大分類`、`動画形式`）は欠落を許容しない。
- 種別名と付与ルールの正本は本表とし、[[RQ-GL-005|タグ辞書]]および[[RQ-GL-008|tag_master]]へ反映する。
