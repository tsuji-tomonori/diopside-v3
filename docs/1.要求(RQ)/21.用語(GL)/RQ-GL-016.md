## 定義
| 項目 | 内容 |
|---|---|
| 用語ID | `RQ-GL-016` |
| 用語名 | [[RQ-GL-016|コメント密度波形]] |
| 英名 | `comment_density_waveform` |
| 定義 | 公開チャット由来のメッセージ時刻分布から算出したコメント密度を、時間軸上に音波状で可視化したデータおよび表示表現。 |
| 判定条件/適用範囲 | 動画[[DD-APP-UI-004|詳細モーダル]]で、[[RQ-GL-015|盛り上がり区間]]の把握と該当秒遷移に使う可視化要素として適用する。 |

## 利用ルール
- 波形は動画開始からの経過秒に対する密度値を連続系列として扱う。
- 密度が高い区間は [[RQ-GL-015|盛り上がり区間]] として抽出・強調表示する。
- 文書・実装・テストで同じ意味で使用する。
