## 詳細仕様
- 本画面は [[DD-UI-009|UI-A04]] として、差分確認とタグ更新確定を同一導線で扱う。
- 外部LLM支援は補助機能として扱い、反映可否の最終判定は管理者操作で確定する。

## 画面構成
| ブロック | 表示内容 | 操作 |
| --- | --- | --- |
| 差分サマリ | 追加/更新/削除件数 | 詳細表示 |
| 動画差分一覧 | 動画単位の差分理由 | 対象選択 |
| 提案入力 | LLM向け入力テキスト | コピー |
| 取込検証 | JSON検証結果、行番号 | 再アップロード |
| 反映操作 | 確認ダイアログ、実行結果 | 反映確定 |

## 状態遷移
- `baseline_loading`: 配信前後データ比較中。
- `reviewing`: 差分確認中。
- `import_validating`: JSON検証中。
- `applying`: タグ反映実行中。
- `applied/error`: 反映成功または失敗。

## 操作制約
- `schemaVersion` と必須キー検証成功まで反映確定を有効化しない。
- 差分未確認項目が残る場合は確定ダイアログで警告を表示する。
- 反映成功後は [[DD-UI-010|UI-A06]] への遷移導線を必須表示する。

## I/Oまたは責務
- 入力: 差分結果、提案入力、取込JSON、[[DD-API-012]]/[[DD-API-013]] 応答。
- 出力: タグ反映要求、検証エラー一覧、配信反映遷移イベント。
