## 設計方針
- 検索入力、選択入力、レンジ入力を同一フォーム群として統一し、条件再入力を最小化する。
- スライダーは2本レンジ（最短/最長）を標準とし、現在値を常時可視化する。

## 設計要点
- `SearchInput`
  - サジェスト開閉とキーボード操作（上下/Enter/Escape）を標準化。
- `SelectField`
  - タグモード、並び順の選択を担当。
- `RangeSlider`
  - 最短値は最長値を超えない。
  - 最長値は最短値を下回らない。
  - 0〜最大は「未指定」として扱う。

## 受入条件
- スライダー値は文言で併記し、色やつまみ位置のみで意味を伝えない。
- 条件確定後、同一セッション内で初期値復元できる（[[RQ-UX-013-01]]）。
- ヘルプ文言は主要画面間で名称/配置を統一する（[[RQ-UX-015-01]]）。
